# AlgoLab Website

This is a **completely standalone** website for AlgoLab materials and solutions.

## Structure

```
algolab/
├── index.html          # Main entry point
├── css/
│   └── style.css      # Independent styling
├── js/
│   └── main.js        # Independent scripts
├── assets/            # Images, files, etc.
└── README.md          # This file
```

## Features

- Fully independent from the main portfolio site
- No shared navigation, headers, or styling
- Can have its own design system and structure
- Accessible via: https://landigf.github.io/algolab/

## Development

This subfolder operates as its own mini-website within the repository.
All resources are self-contained and don't depend on parent directory files.
