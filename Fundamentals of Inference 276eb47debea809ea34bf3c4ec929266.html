Challenges with high-dimensional distributions

So we have problems when dealing with high-dimensional distributions:

  1. Representation (parametrization): imagine nnn﻿ Bernoulli → 2n−12^n - 12n−1﻿ parameters
  1. Learning (estimation) → we need to many data points, we need assumptions
  1. Inference (prediction) → marginalization is prohibitory in high dimensions

So we deal with multivariate gaussian distributions. Why ?

  • d−dimensionald-dimensional d−dimensional﻿ gaussian just needs a vector of d-mean and a covariance matrix → O(d2)O(d^2)O(d2)﻿ parameters
  • many operations (marginalization, conditioning) have closed form → easy inference.
